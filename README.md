> color naming library

## Table of Contents

- [Install](#install)
- [Usage](#usage)
- [License](#license)

## Install

[npm][]:

```sh
npm install coloraze
```

[yarn][]:

```sh
yarn add coloraze
```

## Usage

```js
const Coloraze = require('coloraze');

const coloraze = new Coloraze();

console.log(coloraze.renderName());
// script
```
